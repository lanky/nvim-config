# nvim configuration

a hand-crafted (read: stolen and adapted from other places) neovim configuration, after lazyvim got a bit too much for me

uses [lazy.nvim](https://github.com/folke/lazy.nvim) for plugin management, plus a number of other plugins, some of which will be listed below:

## plugins

### Completion and convenience

- [nvim-autopairs](https://github.com/windwp/nvim-autopairs)
- [mini-surround](https://github.com/nvim-mini/mini.surround)
- [lazydev.nvim](https://github.com/folke/lazydev.nvim)
- [conform.nvim](https://github.com/stevearc/conform.nvim)
- [which-key](https://github.com/folke/which-key.nvim)
- [blink.cmp](https://github.com/saghen/blink.cmp)
- [vim-sensible](https://github.com/tpope/vim-sensible)
- [vim-sleuth](https://github.com/tpope/vim-sleuth)

### LSP support

- [mason](https://github.com/mason-org/mason.nvim)
- [mason-lspconfig](https://github.com/mason-org/mason-lspconfig.nvim)
- [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)
- [nvim-treesitter-textobjects](https://github.com/nvim-treesitter/nvim-treesitter-textobjects)
- [nvim-treesitter-context](https://github.com/nvim-treesitter/nvim-treesitter-context)

### diagnostics

- [trouble.nvim](https://github.com/folke/trouble.nvim)
- [rainbow-delimiters](https://github.com/HiPhish/rainbow-delimiters.nvim)

## UI

- [noice](https://github.com/folke/noice.nvim)
- [snacks.nvim](https://github.com/folke/snacks.nvim)
- [nui.nvim](https://github.com/MunifTanjim/nui.nvim)
- [nvim-notify](https://github.com/rcarriga/nvim-notify)
- [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim)
